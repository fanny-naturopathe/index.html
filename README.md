## Hébergement

Site statique hébergé sur **Netlify**, connecté à ce dépôt GitHub.
Chaque modification poussée sur la branche principale déclenche un redéploiement automatique.

Aucune dépendance, aucun build : HTML/CSS/JS pur, images intégrées en base64.
